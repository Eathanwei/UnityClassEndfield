BossSpine：boss受擊判定的部位。因為boss體積大、動作大還有飛行狀態，所以角色攻擊boss的擊中判定，需要身體、脖子、頭的三個collider。然後再把抓取資料、扣血等等的目標轉換成boss本體。  
BossState：
- 地面狀態：(攻擊冷卻->普攻->攻擊冷卻->技能一->攻擊冷卻->普攻->攻擊冷卻->技能二->)的迴圈行為模式，在攻擊冷卻時會跟隨並面向玩家。
- 在冷卻完成時會判定，玩家是否在普攻或技能的攻擊範圍內，如果不在範圍則延遲攻擊，移動到適當位置再攻擊。技能的判定範圍>普攻的判定範圍>跟隨完成的判定範圍。
- 飛行狀態：半血後會切換成飛行狀態，會召喚小怪，並定時施放火雨技能。
- 減傷護盾：擊倒所有小怪後，會降落切換成三階段，並有減傷護盾的機制，需要收集藍色光球，來削減護盾。
- 其他：進戰鬥時，甦醒吼叫。被角色戰技跟終結技擊中時，結束攻擊跟技能，做受擊動作。死亡時，結束所有其他動作跟特效，做死亡動作。

DragonBreath：開啟噴火特效，與噴火攻擊判定。  
MarbleEffect：圓形範圍會根據數量切分成扇形，每個藍色光球在扇形範圍的隨機位置。藍色光球被角色觸碰時，會播放獲得的特效，並減弱護盾效果。  
MarbleManager：統計藍色光球子物件的數量，分配編號，並通知各藍色光球開啟。  
MeteorsEffect：圓形範圍會根據數量切分成扇形，每個火雨在扇形範圍的隨機位置。火雨會先預警範圍數秒讓玩家躲避，然後才開始播放火雨特效，並做火雨攻擊判定。  
MeteorsEffectPlayer：會有一個火雨的中心固定在，以玩家為中心的半徑範圍內的隨機位置，避免其他完全隨機的火雨會覆蓋不到玩家，讓玩家置身事外。  
MeteorsManager：統計火雨子物件的數量，分配編號，並通知各火雨開啟。

CharacterBasicData: 角色基本數值，例如生命值、攻擊力等等。也包含一些沒有用到的，例如角色的等級、天賦、潛能等等。
CharacterCombatData: 角色的統整數值，角色基本數值加上武器、裝備，包含防禦力、抗性、爆擊、屬性增傷、攻擊種類增傷、治療加成、連攜技冷卻減少、終結技技力獲取效率加成等等。
CharacterCombatLive: 角色的即時戰鬥數值，加上技力獲取、連攜技冷卻、技能釋放、主控角色普攻、普攻計數、角色受擊等等。
CombatUnit: 戰鬥單位基本數值，包括生命值、攻擊力、防禦力、抗性。四種屬性的法術附著跟debuff效果、法術/物理脆弱。傷害乘區計算、治療乘區計算。通知UI更新。
EnemyCombatData: 基本數值設定。
Weapon: 武器詞條、角色更換武器的數值更新。

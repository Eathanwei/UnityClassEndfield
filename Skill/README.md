CharacterSkill: 攻擊與技能種類架構。連攜技觸發架構。  
CombatEventManager: 戰鬥相關Event的管理，用來觸發連攜技。  

ElectricSword: 電屬性劍士  
- 連攜技觸發(AbleCombo)：消耗debuff觸發，包含重劍手連攜技碎冰、槍手戰技消耗燃燒爆擊。  
- 普攻、重擊：重擊回復14點戰技。  
- 連攜技(combo)：回復15點戰技，如果敵人身上有導電，則額外消耗導電 15+5*導電層數 點戰技。  
- 戰技(short)：添加電磁附著。  
- 終結技(long)：回復60點戰技，添加1層導電。

FirePistol: 火屬性槍手  
- 連攜技觸發(AbleCombo)：敵人被添加附著觸發，包含角色戰技、法師終結技。  
- 普攻、重擊：重擊回復14.4點戰技。  
- 連攜技(combo)：添加灼熱附著。  
- 戰技(short)：如果敵人身上有燃燒，則消耗燃燒獲得10秒的必定爆擊，如果沒有燃燒，則添加灼熱附著。  
- 終結技(long)：添加1層燃燒。
  
IceGreatSword: 冰屬性重劍手  
- 連攜技觸發(AbleCombo)：敵人被添加冰凍觸發，包含戰技的寒冷附著將其他附著轉換成冰凍、終結技的冰凍。  
- 普攻、重擊：重擊回復10點戰技。  
- 連攜技(combo)：施加猛擊，觸發碎冰，消耗冰凍造成大量物理傷害。猛擊在敵人沒有破防時，會造成1層破防，有破防時，則會消耗所有破防層數造成大量物理傷害。破防相當於物理附著。  
- 戰技(short)：添加寒冷附著。  
- 終結技(long)：添加1層冰凍。
 
NatureMagic: 自然屬性法師  
- 連攜技觸發(AbleCombo)：主控角色打出重擊時觸發。  
- 普攻、重擊：重擊回復20.2點戰技。  
- 連攜技(combo)：回復生命值，添加1層腐蝕。  
- 戰技(short)：回復生命值，添加自然附著。  
- 終結技(long)：回復生命值，添加22%法術脆弱，添加當前敵人附著。

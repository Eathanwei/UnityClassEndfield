CharacterCombatSO: 角色基本數值SO
HW2_camera: 攝影機控制，以Cinemachine Free Look Camera的邏輯為基礎，自行撰寫。
            利用平面角度跟垂直角度的旋轉，決定攝影機的方向，再加上避障功能跟Damping過渡去處理攝影機的距離。
InputManager: 玩家輸入管理，包括主控角色操控、切換主控角色、角色技能釋放、連攜技順序管理。最新觸發的連攜技會最後施放。
LocomotionBlend: 主控角色的移動、旋轉、鎖敵。移動狀態由WASD移動跟旋轉，攻擊與技能狀態會旋轉面對鎖定的敵人，閃避與受擊狀態不能移動跟旋轉。
                主控角色的鎖敵，會比較原本鎖定的敵人跟距離最近的敵人，如果原本鎖定敵人的距離減最近的敵人的距離，超過緩衝值，就切換鎖定的敵人為距離最近的敵人。
PlaceManager: 戰鬥狀態控管。如果主控角色進入敵人群的偵測範圍，非主控角色與敵人都會進入戰鬥狀態；如果主控角色離開偵測範圍或是打倒敵人群，非主控角色與敵人都會離開戰鬥狀態。
              離開戰鬥的偵測範圍會比進入戰鬥的偵測範圍大，避免頻繁切換。boss的戰鬥無法離開。
              非主控角色的鎖敵，只會鎖敵觸發戰鬥的敵人，普通攻擊的鎖敵邏輯同LocomotionBlend，技能則優先打主控角色鎖定的敵人。
WeaponSO: 武器種類與詞條。
